# Changelog

## panelsummary 0.1.3

CRAN release: 2026-03-20

- `clean_raw` now includes a `booktabs` and `linesep` argument that will
  be passed directly into
  [`kableExtra::kbl`](https://rdrr.io/pkg/kableExtra/man/kbl.html)
- Small bug fixes for `fixest` attribute names introduced in `fixest`
  v0.14.0 (Thank you lrberge.)

## panelsummary 0.1.2

CRAN release: 2023-08-13

- New `pretty_num` argument for large numbers with commas.

## panelsummary 0.1.1

CRAN release: 2023-03-15

- Updated to fix `modelsummary` 1.3 breaking change update. Package now
  depends on `modelsummary` version 1.3 and above.

## panelsummary 0.1.0

CRAN release: 2022-12-20

- Added a `NEWS.md` file to track changes to the package.
